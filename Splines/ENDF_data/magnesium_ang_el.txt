This evaluated data file is based on a software system built      1225 1451  246
around the nuclear model code TALYS [kon07], version 2.0. It is   1225 1451  247
part of a large collection of isotopic evaluations, all created by1225 1451  248
running the system with either adjusted or default input          1225 1451  249
parameters. This means that the data in the entire collection have1225 1451  250
been tested in detail for many isotopes against individual        1225 1451  251
experimental data, while for other isotopes it is only as good as 1225 1451  252
the global quality of TALYS at the moment. The mutual quality of  1225 1451  253
all individual data files is however consistent: The same set of  1225 1451  254
nuclear models is used and, equally important, the same ENDF-6    1225 1451  255
formatting procedures for each isotope. The data file has been    1225 1451  256
created automatically using the ENDF-6 format generator TEFAL.    1225 1451  257
                                                                  1225 1451  258

nuclear data needed for transport, activation, damage, heating,   1225 1451  260
radioactivity, and shielding applications over the incident       1225 1451  261
proton energy range from 0.25 to 200. MeV.                        1225 1451  262
                                                                  1225 1451  263

of nuclear reactions. TALYS simulates reactions that involve      1225 1451  279
neutrons, gamma-rays, protons, deuterons, tritons, helions and    1225 1451  280
alpha-particles, in the 1 keV - 200 MeV energy range. This is     1225 1451  281

theoretical analysis that utilizes the optical model, compound    1225 1451  284
nucleus statistical theory, fission, direct reactions and         1225 1451  285
