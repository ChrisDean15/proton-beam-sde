This evaluated data file is based on a software system built      1125 1451  178
around the nuclear model code TALYS [kon07], version 2.0. It is   1125 1451  179
part of a large collection of isotopic evaluations, all created by1125 1451  180
running the system with either adjusted or default input          1125 1451  181
parameters. This means that the data in the entire collection have1125 1451  182
been tested in detail for many isotopes against individual        1125 1451  183
experimental data, while for other isotopes it is only as good as 1125 1451  184
the global quality of TALYS at the moment. The mutual quality of  1125 1451  185
all individual data files is however consistent: The same set of  1125 1451  186
nuclear models is used and, equally important, the same ENDF-6    1125 1451  187
formatting procedures for each isotope. The data file has been    1125 1451  188
created automatically using the ENDF-6 format generator TEFAL.    1125 1451  189
                                                                  1125 1451  190
The resulting data file provides a complete representation of     1125 1451  191
nuclear data needed for transport, activation, damage, heating,   1125 1451  192
radioactivity, and shielding applications over the incident       1125 1451  193
proton energy range from 0.25 to 200. MeV.                        1125 1451  194
                                                                  1125 1451  195
All transport and activation data for neutrons, charged particles,1125 1451  196
photons and residual nuclides are filed. This includes cross      1125 1451  197
sections, angular distributions, double-differential spectra,     1125 1451  198
isomeric production, discrete and continuum photon production     1125 1451  199
cross sections, residual production cross sections and recoils.   1125 1451  200
This general purpose evaluation can thus be used as both          1125 1451  201
transport and activation library.                                 1125 1451  202

***************************** T A L Y S **************************1125 1451  208
                                                                  1125 1451  209
TALYS is a computer code system for the prediction and analysis   1125 1451  210
of nuclear reactions. TALYS simulates reactions that involve      1125 1451  211
neutrons, gamma-rays, protons, deuterons, tritons, helions and    1125 1451  212
alpha-particles, in the 1 keV - 200 MeV energy range. This is     1125 1451  213
achieved by implementing a suite of nuclear reaction models into  1125 1451  214
a single code system. This nuclear data file is based on a        1125 1451  215
theoretical analysis that utilizes the optical model, compound    1125 1451  216
nucleus statistical theory, fission, direct reactions and         1125 1451  217
pre-equilibrium processes, in combination with databases and      1125 1451  218
models for nuclear structure. The following output of TALYS is    1125 1451  219
stored in this data file:                                         1125 1451  220
                                                                  1125 1451  221
- Elastic and non-elastic cross sections                          1125 1451  222
- Elastic scattering angular distributions                        1125 1451  223
- Recoil data                                                     1125 1451  224
- Total particle cross sections, e.g. (p,xn), (p,xp),..           1125 1451  225
- Total particle energy spectra                                   1125 1451  226
- Total particle double-differential spectra                      1125 1451  227
- Isomeric and ground state cross sections                        1125 1451  228
- Residual production cross sections                              1125 1451  229
                                                                  1125 1451  230
For a description of the used nuclear models and their            1125 1451  231
parameterization we refer to [kon19]. If for this specific        1125 1451  232
nuclide we deviate from the global parameterization, it is made   1125 1451  233
explicit above under Nuclide Specific Information.                1125 1451  234

##### MF33: COVARIANCES OF NEUTRON CROSS SECTIONS                 1125 1451  304
                                                                  1125 1451  305
In MF33, we store the covariances of most, but not all, reaction  1125 1451  306
channels that are given in MF3. All covariances are generated     1125 1451  307
with the TASMAN code [kon19], which generates TALYS runs using    1125 1451  308
a Monte Carlo parameter method.                                   1125 1451  309
For each MT-number, the information consists of:                  1125 1451  310
1. Covariances for MTa, Ea and MTa, Ea (variances)                1125 1451  311
2. Covariances for MTa, Ea and MTa, Eb (correlations for the      1125 1451  312
   same reaction channel between different energies)              1125 1451  313
3. Covariances for MTa, Ea and MTb, Ea (correlations between      1125 1451  314
   different reaction channels for the same energy)               1125 1451  315
4. Covariances for MTa, Ea and MTb, Eb (correlations between      1125 1451  316
   different reaction channels for different energies)            1125 1451  317
Classes 1 and 2 are covered by NI-type subsubsections with        1125 1451  318
flag LB=5, followed by a LB=8 section which contains              1125 1451  319
the cross section variances multiplied by 1.e-3. Next, classes 3  1125 1451  320
and 4 are covered by flag LB=6 to account for the different       1125 1451  321
energy grids that two correlating MT-sections may have. Since     1125 1451  322
the total correlation matrix is fully symmetric, we only need to  1125 1451  323
include the correlation elements between different MT-numbers     1125 1451  324
once, and we do that in the subsection with the lowest of the     1125 1451  325
two MT-numbers. We include covariance information for all the     1125 1451  326
total cross sections, the lumped partial cross sections and the   1125 1451  327
individual inelastic cross sections per level (MT51,...). We do   1125 1451  328
not include covariances for MT600 and higher MT-numbers. For the  1125 1451  329
most important channels, we include cross-channel correlations.   1125 1451  330
