This evaluated data file is based on a software system built      1625 1451  149
around the nuclear model code TALYS [kon07], version 2.0. It is   1625 1451  150
part of a large collection of isotopic evaluations, all created by1625 1451  151
running the system with either adjusted or default input          1625 1451  152
parameters. This means that the data in the entire collection have1625 1451  153
been tested in detail for many isotopes against individual        1625 1451  154
experimental data, while for other isotopes it is only as good as 1625 1451  155
the global quality of TALYS at the moment. The mutual quality of  1625 1451  156
all individual data files is however consistent: The same set of  1625 1451  157
nuclear models is used and, equally important, the same ENDF-6    1625 1451  158
formatting procedures for each isotope. The data file has been    1625 1451  159
created automatically using the ENDF-6 format generator TEFAL.    1625 1451  160
                                                                  1625 1451  161
The resulting data file provides a complete representation of     1625 1451  162
nuclear data needed for transport, activation, damage, heating,   1625 1451  163
radioactivity, and shielding applications over the incident       1625 1451  164
proton energy range from 0.25 to 200. MeV.                        1625 1451  165
                                                                  1625 1451  166
All transport and activation data for neutrons, charged particles,1625 1451  167
photons and residual nuclides are filed. This includes cross      1625 1451  168
sections, angular distributions, double-differential spectra,     1625 1451  169
isomeric production, discrete and continuum photon production     1625 1451  170
cross sections, residual production cross sections and recoils.   1625 1451  171
This general purpose evaluation can thus be used as both          1625 1451  172

TALYS is a computer code system for the prediction and analysis   1625 1451  181
of nuclear reactions. TALYS simulates reactions that involve      1625 1451  182
neutrons, gamma-rays, protons, deuterons, tritons, helions and    1625 1451  183
alpha-particles, in the 1 keV - 200 MeV energy range. This is     1625 1451  184
achieved by implementing a suite of nuclear reaction models into  1625 1451  185
a single code system. This nuclear data file is based on a        1625 1451  186
theoretical analysis that utilizes the optical model, compound    1625 1451  187
nucleus statistical theory, fission, direct reactions and         1625 1451  188
pre-equilibrium processes, in combination with databases and      1625 1451  189
models for nuclear structure. The following output of TALYS is    1625 1451  190
stored in this data file:                                         1625 1451  191
                                                                  1625 1451  192
- Elastic and non-elastic cross sections                          1625 1451  193
- Elastic scattering angular distributions                        1625 1451  194
- Recoil data                                                     1625 1451  195
- Total particle cross sections, e.g. (p,xn), (p,xp),..           1625 1451  196
- Total particle energy spectra                                   1625 1451  197
- Total particle double-differential spectra                      1625 1451  198
- Isomeric and ground state cross sections                        1625 1451  199
- Residual production cross sections                              1625 1451  200

##### MF33: COVARIANCES OF NEUTRON CROSS SECTIONS                 1625 1451  275
                                                                  1625 1451  276
In MF33, we store the covariances of most, but not all, reaction  1625 1451  277
channels that are given in MF3. All covariances are generated     1625 1451  278
with the TASMAN code [kon19], which generates TALYS runs using    1625 1451  279
a Monte Carlo parameter method.                                   1625 1451  280
For each MT-number, the information consists of:                  1625 1451  281
1. Covariances for MTa, Ea and MTa, Ea (variances)                1625 1451  282
2. Covariances for MTa, Ea and MTa, Eb (correlations for the      1625 1451  283
   same reaction channel between different energies)              1625 1451  284
3. Covariances for MTa, Ea and MTb, Ea (correlations between      1625 1451  285
   different reaction channels for the same energy)               1625 1451  286
4. Covariances for MTa, Ea and MTb, Eb (correlations between      1625 1451  287
   different reaction channels for different energies)            1625 1451  288
Classes 1 and 2 are covered by NI-type subsubsections with        1625 1451  289
flag LB=5, followed by a LB=8 section which contains              1625 1451  290
the cross section variances multiplied by 1.e-3. Next, classes 3  1625 1451  291
and 4 are covered by flag LB=6 to account for the different       1625 1451  292
energy grids that two correlating MT-sections may have. Since     1625 1451  293
the total correlation matrix is fully symmetric, we only need to  1625 1451  294
include the correlation elements between different MT-numbers     1625 1451  295
once, and we do that in the subsection with the lowest of the     1625 1451  296
two MT-numbers. We include covariance information for all the     1625 1451  297
total cross sections, the lumped partial cross sections and the   1625 1451  298
individual inelastic cross sections per level (MT51,...). We do   1625 1451  299
not include covariances for MT600 and higher MT-numbers. For the  1625 1451  300
most important channels, we include cross-channel correlations.   1625 1451  301
